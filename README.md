# homebrew-opencode

Homebrew tap for the [cwirz/opencode](https://github.com/cwirz/opencode) fork
(compact tool schemas + lazy MCP tool loading).

## Install

```sh
brew install cwirz/opencode/opencode-cwirz
```

Installs as `opencode-cwirz`, coexists with the official `opencode`.
Optionally: `alias opencode=opencode-cwirz`.
